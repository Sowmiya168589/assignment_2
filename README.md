SecureCheck: A Python-SQL Digital Ledger for Police Post Logs
SecureCheck is a Python–SQL based digital ledger designed to modernize police check post operations. It enables centralized logging, real-time analytics, and risk-based vehicle identification using an interactive Streamlit dashboard. The system processes traffic stop data, stores it in an SQL database, and provides insights such as arrest trends, peak stop hours, and high-risk vehicles to support data-driven law enforcement decisions.
